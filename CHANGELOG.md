# Changelog

## v0.1.0 – Initial Release

### Features
- useStableAsync – safe async handling
- useDebouncedValue – debounce values
- useIsMobile – mobile detection with ResizeObserver
- useClickOutside – outside click handling
- useCountdown – countdown timer
- useLocalStorage – persistent state with cross-tab sync
- usePing – network status & latency
- useLatestRef – avoid stale closures
- usePreviousValue – track previous values
- useMounted – safe mounted checks
- useOnceEffect – run effects once
